# Functions

Storage of R functions that were created in support of this stock assessment workflow.
having this strucure can encourage reproducbility and find functions quickly. Please
only include one function or a set of related functions in a file rather than a
singluar large files of functions. 

By storing functions in this format, you will be able to load all of these
functions into your environment using `devtools::load_all()` instead of either
running each individually or sourcing files.
